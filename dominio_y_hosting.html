Dominio y Hosting

Elegir un dominio

Una vez hemos elegido la temática de la web debemos elegir el dominio. Podemos comprarlo en lugares como DonDominio, donde consultaremos si el dominio que queremos está libre. 

Un dominio web se divide en dos partes principales: el nombre de dominio y la extensión del dominio. A continuación, explicaré cada una de estas partes:

Nombre de dominio: Es la parte principal y más distintiva de un dominio web. Este nombre es único para cada sitio web y generalmente está relacionado con el nombre de la empresa, organización o el propósito del sitio web. Por ejemplo, en el dominio "google.com", "google" es el nombre de dominio. Los nombres de dominio pueden contener letras, números y guiones, pero no pueden contener espacios ni caracteres especiales.
Extensión de dominio: También conocida como TLD (Dominio de Nivel Superior), es la parte final del dominio que sigue al nombre de dominio. La extensión de dominio indica la categoría o tipo de sitio web al que pertenece. Algunos ejemplos comunes de extensiones de dominio son ".com", ".org", ".net", ".edu", ".gov", entre otros. Cada extensión de dominio tiene su propio propósito y está destinada a diferentes tipos de organizaciones o ubicaciones geográficas.

Elegir el nombre

Aunque cada vez tiene menos importancia para el SEO, podemos tratar de que el dominio se parezca a la palabra clave para la que queremos posicionar, pero en muchos casos está ocupada.

Elegir extensión

Existe el falso mito de que hay extensiones que se posicionan mejor que otras y no es así. Debemos elegir la extensión excluyendo los que están asociados a una zona como .es y por otro lado comparando precios. Tened cuidado con algunas extensiones que cobran poco el primer año y aumenta muchísimo al siguiente. Personalmente recomiendo .com .org o .net .

Además debemos entrar antes de comprar un dominio a archive.org y buscar si el dominio que vamos a comprar ha sido utilizado en un pasado y tratar de identificar uno que sea "vírgen" para evitar determinados problemas que nos puedan surgir.

Elegir un Hosting

Ahora una vez hemos comprado un dominio debemos alojarlo en un hosting, que podremos elegir entre distintas empresas. Actualmente los más famosos son Hostinger o Hostalia. Además hay numerosas ofertas donde al contratar el hosting se nos regala un dominio. Después de ello debemos alojar nuestro dominio en el hosting, os dejo aquí un tutorial.

Aclarar que los dominios y servidores gratis existen pero no son eficientes a la hora de aplicar SEO por su bajo rendimiento y velocidad.
